# Лабораторна робота 2

Цей проєкт — це повноцінний односторінковий веб-застосунок (SPA) інтернет-магазину, розроблений на Node.js, Express та React. Додаток реалізує цикл покупок: від перегляду каталогу і роботи з кошиком до безпечного оформлення замовлення з вказівкою адреси доставки. Для користувачів створено систему реєстрації та авторизації на базі JWT-токенів із доступом до особистого кабінету. У профілі автоматично оновлюються статуси замовлень у реальному часі та працює алгоритм персоналізованих рекомендацій на основі історії покупок, а для швидкого завантаження налаштовано GZIP-компресію даних.

## Як запустити проєкт

**1. Запуск сервера (Backend)**
Відкрийте термінал у папці з серверним кодом та виконайте команди:
npm install
node index.js

**2. Запуск клієнта (Frontend)**
Відкрийте термінал у папці з кліентським кодом та виконайте команди:
npm install
npm run dev
